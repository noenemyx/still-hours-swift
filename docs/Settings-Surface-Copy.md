## 구현 참고

- `Still Hours는` / `Still Hours is` — 섹션 헤더, 탭 타이틀에 사용
- `———————————` — U+2014 (em dash) × 11, 디자인상 구분선
- `[라이브러리 내보내기 (JSON)]` — tappable row, 아이콘 없음
- `데이터는 당신의 iCloud에만 존재합니다.` — caption, secondary color
- `[블로그]` — tappable, URL은 개발 단계에서 확정
- Section 순서: Identity → Data Sovereignty → Made by (변경 시 Design.md §5.6 동기화 필요)
