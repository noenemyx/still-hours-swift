# Still Hours — プライバシーポリシー

最終更新日：2026年5月22日  
施行日：2026年5月22日  
適用バージョン：Still Hours v1.0 以降

---

## はじめに

Still Hours は、あなたの所有物にまつわる記憶を記録するためのツールです。このポリシーでは、Still Hours がユーザーの情報をどのように扱うか、そして扱わないかを明確にご説明します。お読みいただくのに5分ほどかかります。

---

## §1 収集する情報

Still Hours にはサーバーがありません。開発者（ownlifelab）はユーザーのデータにアクセスできません。

**デバイス上のデータ**  
アプリ内で入力するすべてのコンテンツ — アイテム、メモリー、コレクション、添付ファイル — はユーザーのデバイスにのみ保存されます。

**iCloud 同期（任意設定、v1.1 以降）**  
ユーザーが設定から iCloud 同期を明示的に有効にした場合のみ、データは Apple CloudKit プライベートデータベースに書き込まれます。このデータベースはユーザーの Apple ID に紐付けられた非公開領域であり、ownlifelab は読み取り・書き込み・閲覧のいずれのアクセス権も持ちません。v1.0 ではデフォルトで iCloud 同期はオフになっています。

**メタデータの外部取得**  
書籍や音楽アルバムなどのメタデータを自動取得する際、アプリはその作品の識別子（例：ISBN）のみを外部 API に送信します。ユーザーの氏名、位置情報、デバイス情報、アカウント情報は送信されません。

**Apple が収集する情報**  
App Store での購入、ダウンロード、クラッシュレポートは Apple のシステムが処理します。このデータには Apple のプライバシーポリシーが適用されます。ownlifelab はこのデータを受け取りません。

---

## §2 行わないこと

以下の項目は Still Hours の Promise 5条項に基づく恒久的な約束であり、現在および将来にわたって適用されます。

**トラッキングなし**  
広告識別子（IDFA）、広告 SDK、サードパーティ分析ツール、ユーザー行動トラッキングスクリプトは使用しません。Apple の `PrivacyInfo.xcprivacy` マニフェストに `NSPrivacyTracking = false` として宣言されています。アプリ内の 設定 → Data Privacy でご確認いただけます。

**広告なし**  
アプリ内に広告はありません。Apple Search Ads を含む外部広告チャネルも利用しません。

**データの販売なし**  
ユーザーの情報を第三者に販売・提供・譲渡することはありません。

**公開プロフィールなし**  
ユーザーアカウント、公開フィード、ソーシャルグラフはありません。すべてのコレクションはデフォルトで非公開です。

**AI による判断なし**  
AI がユーザーの記録を評価・推薦・自動分類することはありません。バーコードスキャン後のメタデータ補完（OCR・画像認識）などの AI 補助機能は、ユーザーが明示的に呼び出した場合にのみ動作します。これらの処理結果がモデルの学習に使用されることはありません。

**サブスクリプションなし**  
Still Hours は買い切り型の有料アプリです。定期課金はありません。

## §3 サードパーティサービス

Still Hours が依存する外部サービスは以下のとおりです。

| サービス | 提供者 | 目的 | ユーザーデータの送信 |
|---------|------|------|-----------------|
| iCloud / CloudKit | Apple Inc. | 任意のデバイス間同期 | ユーザーが同期を有効化した場合のみ、Apple の暗号化されたプライベート DB に保存 — ownlifelab はアクセス不可 |
| App Store | Apple Inc. | アプリ配布および購入処理 | Apple の標準ポリシーが適用 |
| 書籍・メディアメタデータ API | 各提供者 | アイテムメタデータの自動補完 | 作品識別子のみ（例：ISBN）— ユーザー識別情報は含まない |

上記以外のサードパーティ SDK、広告ネットワーク、分析プラットフォームは含まれていません。

---

## §4 データ主権（Data Sovereignty）

Still Hours の核となる約束のひとつは、ユーザーがいつでも自分のデータを取り出せるということです。

**エクスポート形式**  
設定 → Data Sovereignty から、ライブラリ全体を JSON・CSV・PDF 形式でエクスポートできます。エクスポートされたファイルはプレーンテキストであり、Still Hours がなくても開くことができます。

**アプリを削除した場合**  
アプリを削除するとデバイス上のデータが削除されます。iCloud 同期を使用していた場合、CloudKit に保存されたデータは Apple の iCloud 設定から別途削除できます。

**サービス終了時**  
Still Hours が終了する場合でも、すでにエクスポートしたデータはユーザーが引き続き保持します。サービス終了の少なくとも 30 日前に告知し、その期間中エクスポート機能を維持します。

**ownlifelab によるアクセスなし**  
ユーザーのデータが ownlifelab のサーバーに保存されたり送信されたりすることはありません。CloudKit のデータにはユーザーの Apple ID でのみアクセスできます。

---

## §5 子どものプライバシー

Still Hours は App Store レーティング 4+（全年齢対象）でリリースされます。本アプリは 13 歳未満の子どもを対象としたサービスを運営しておらず、13 歳未満の子どもから意図的に個人情報を収集することはありません。

保護者の方で、13 歳未満のお子様が本アプリを使用していることを確認し、ご懸念がある場合は §8 の連絡先までお知らせください。

---

## §6 国際的なデータ移転

Still Hours は独自の国際的なデータ移転を行いません。データはユーザーのデバイスに保存されます。iCloud 同期が有効な場合は、ユーザーの Apple ID に紐付けられた Apple の iCloud インフラストラクチャで処理されます。iCloud データの処理場所については Apple のプライバシーポリシーをご参照ください。

メタデータ API への問い合わせでは、作品識別子のみが該当サービスのサーバーに送信されます。これらのリクエストにはユーザーを特定できる情報は含まれません。

---

## §7 ユーザーの権利

以下の権利は、ownlifelab に連絡することなく、アプリ内で直接行使できます。

**閲覧**  
すべてのデータはアプリ内でいつでも確認できます。

**エクスポート**  
設定 → Data Sovereignty → エクスポート（JSON / CSV / PDF）。アカウント不要、承認不要、即時実行可能です。

**削除**  
個々のアイテム・メモリー・コレクションはアプリ内で直接削除できます。全データを削除するには、アプリを削除（デバイス上のデータ）し、Apple の iCloud 設定 → ストレージを管理 → Still Hours（CloudKit データ）から削除してください。

**修正**  
すべてのデータはアプリ内で直接編集できます。

アプリはこれらすべての権利をご自身で行使できるよう設計されています。サポートが必要な場合は §8 の連絡先までご連絡ください。
