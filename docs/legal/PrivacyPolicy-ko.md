# Still Hours — 개인정보 처리방침

최종 수정일: 2026년 5월 22일  
발효일: 2026년 5월 22일  
적용 버전: Still Hours v1.0 이상

---

## 들어가며

Still Hours는 기억을 기록하는 도구입니다. 이 방침은 Still Hours가 사용자의 정보를 어떻게 다루는지 — 그리고 다루지 않는지 — 를 명확하게 설명합니다. 읽는 데 5분이면 충분합니다.

---

## §1 수집하는 정보

Still Hours는 서버가 없습니다. 개발자(ownlifelab)는 사용자의 데이터에 접근하지 않습니다.

**기기 내 저장 데이터**  
사용자가 앱 안에 입력하는 모든 항목 — 아이템, 메모리, 컬렉션, 첨부파일 — 은 사용자의 기기에만 저장됩니다.

**iCloud 동기화 (선택, v1.1 이상)**  
사용자가 설정에서 iCloud 동기화를 직접 활성화한 경우에만, 데이터는 Apple CloudKit Private Database에 기록됩니다. 이 데이터베이스는 사용자의 Apple ID에 종속된 비공개 영역이며, ownlifelab은 이에 대한 읽기·쓰기·열람 권한이 없습니다. v1.0은 기본적으로 iCloud 동기화가 꺼져 있습니다.

**메타데이터 외부 조회**  
책·음반 등의 메타데이터를 자동으로 가져올 때, 앱은 해당 작품의 식별자(예: ISBN)만 외부 API에 전달합니다. 사용자의 이름, 위치, 기기 정보, 계정 정보는 전송되지 않습니다.

**Apple이 수집하는 정보**  
App Store 구매, 앱 다운로드, 기기 충돌 보고서는 Apple의 시스템이 처리합니다. 이 데이터는 Apple의 개인정보 처리방침이 적용됩니다. ownlifelab은 이 데이터를 수신하지 않습니다.

---

## §2 하지 않는 것

아래 항목은 Still Hours의 Promise 5조항에 따라 현재뿐 아니라 향후에도 적용되는 영구 약속입니다.

**추적 없음**  
광고 식별자(IDFA), 광고 SDK, 제3자 분석 도구, 사용자 행동 추적 스크립트를 사용하지 않습니다. Apple의 `PrivacyInfo.xcprivacy` 매니페스트에 `NSPrivacyTracking = false`로 선언되어 있으며, 앱 → 설정 → Data Privacy에서 확인할 수 있습니다.

**광고 없음**  
앱 내 광고가 없습니다. Apple Search Ads를 포함한 외부 광고 채널을 통한 사용자 유입 추적도 하지 않습니다.

**데이터 판매 없음**  
사용자의 정보를 제3자에게 판매하거나 이전하거나 공유하지 않습니다.

**공개 프로필 없음**  
사용자 계정, 공개 피드, 소셜 그래프가 없습니다. 모든 컬렉션은 기본적으로 비공개입니다.

**AI 판단 없음**  
AI가 사용자의 기록을 평가하거나 추천하거나 자동 분류하지 않습니다. AI 기술은 바코드 스캔 후 메타데이터 보완(OCR·이미지 인식)과 같이 사용자가 요청한 보조 기능에만 사용되며, 그 결과가 외부로 전송되거나 학습에 사용되지 않습니다.

**구독 없음**  
일회성 유료 앱입니다. 반복 결제 모델이 없습니다.

## §3 제3자 서비스

Still Hours가 의존하는 외부 서비스는 다음과 같습니다.

| 서비스 | 제공사 | 목적 | 사용자 데이터 전송 여부 |
|--------|--------|------|----------------------|
| iCloud / CloudKit | Apple Inc. | 선택적 기기 간 동기화 | 사용자가 직접 활성화 시, Apple의 암호화된 Private DB에 저장 — ownlifelab 미접근 |
| App Store | Apple Inc. | 앱 배포 및 구매 처리 | Apple의 표준 정책 적용 |
| 외부 도서·미디어 API | 각 제공사 | 작품 메타데이터 자동 완성 | 작품 식별자(ISBN 등)만 — 사용자 식별 정보 없음 |

위 세 가지 외에 제3자 SDK, 광고 네트워크, 분석 플랫폼은 포함되어 있지 않습니다.

---

## §4 데이터 주권 (Data Sovereignty)

Still Hours의 핵심 약속 중 하나는 사용자가 자신의 데이터를 언제든 꺼내갈 수 있다는 것입니다.

**내보내기 형식**  
앱 → 설정 → Data Sovereignty에서 JSON, CSV, PDF 형식으로 전체 라이브러리를 내보낼 수 있습니다. 내보낸 파일은 암호화되지 않은 평문이므로 Still Hours 없이도 열 수 있습니다.

**앱 삭제 시**  
기기에서 앱을 삭제하면 기기 내 데이터가 삭제됩니다. iCloud 동기화를 사용하고 있었다면, CloudKit에 저장된 데이터는 Apple의 iCloud 설정에서 별도로 삭제할 수 있습니다.

**서비스 종료 시**  
Still Hours 서비스가 종료되더라도 이미 내보낸 데이터는 사용자가 계속 보유합니다. 서비스 종료 30일 전에 공지하고, 해당 기간 동안 내보내기 기능을 유지합니다.

**ownlifelab의 데이터 접근 없음**  
사용자의 데이터는 ownlifelab의 서버에 저장되거나 전달되지 않습니다. CloudKit Private Database에 저장된 데이터는 사용자의 Apple ID로만 접근 가능합니다.

---

## §5 아동 개인정보

Still Hours는 App Store 등급 4+(전체 이용가)로 출시됩니다. 앱은 만 13세 미만 아동을 대상으로 하는 서비스를 운영하지 않으며, 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.

만 13세 미만 아동의 보호자로서 해당 아동이 앱을 사용하고 있음을 발견하고 우려가 있을 경우, §8의 연락처로 알려주십시오.

---

## §6 국제 데이터 이전

Still Hours는 별도의 국제 데이터 이전을 수행하지 않습니다. 데이터는 사용자의 기기에 저장되며, iCloud 동기화 시에는 Apple의 iCloud 인프라(사용자의 Apple ID와 연결된 데이터 센터)에 저장됩니다. Apple의 iCloud 데이터 처리 위치는 Apple의 개인정보 처리방침에서 확인할 수 있습니다.

외부 메타데이터 API 호출 시, 작품 식별자만 해당 서비스의 서버로 전달됩니다. 이 요청에는 사용자를 식별할 수 있는 정보가 포함되지 않습니다.

---

## §7 사용자의 권리

사용자는 자신의 데이터에 대해 다음 권리를 행사할 수 있습니다. 모든 작업은 앱 내에서 직접 수행할 수 있습니다.

**열람**  
앱 안에서 모든 데이터를 직접 확인할 수 있습니다.

**내보내기**  
앱 → 설정 → Data Sovereignty → 내보내기 (JSON / CSV / PDF). 계정 없이, ownlifelab의 승인 없이 즉시 가능합니다.

**삭제**  
개별 아이템·메모리·컬렉션은 앱에서 직접 삭제할 수 있습니다. 전체 데이터 삭제는 앱 삭제(기기 내 데이터) 및 Apple iCloud 설정 → 저장 공간 관리 → Still Hours (CloudKit 데이터)로 수행할 수 있습니다.

**수정**  
모든 데이터는 앱에서 직접 수정할 수 있습니다.

ownlifelab에 별도 요청 없이 앱 안에서 모든 권리를 직접 행사할 수 있도록 설계되어 있습니다. 도움이 필요한 경우 §8의 연락처로 문의할 수 있습니다.
